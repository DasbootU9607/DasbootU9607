## Featured Projects

### HaLoop

`Next.js` `React` `Tailwind CSS` `SQLite` `Radix UI` `React Flow` `SSE`

Co-leader and full-stack developer for a DevSecOps platform that acts as a safety layer for AI coding agents, intercepting high-risk code changes and enforcing a mandatory human approval workflow.

- Built a high-performance API with Next.js API Routes and SQLite, including a dual-layer risk scoring engine and local-first persistence.
- Engineered real-time reviewer feedback propagation between the web dashboard and VS Code plugin using Server-Sent Events and long-polling.
- Added critical safety mechanisms such as Incident Mode, immutable audit trails, and Blast Radius visualization for impact review.
- Awarded 2nd Place in the OpenAI Track and a Silver Award at Deep Learning Week Hackathon 2026.

### PillowTalk

`Kotlin` `Jetpack Compose` `Navigation Compose` `Material 3` `WorkManager` `SharedPreferences` `Vico`

Solo Android project focused on reducing unplanned night-time phone use through app blocking, usage limits, focus sessions, and structured evening planning.

- Designed ChronoCapsule, a workload-aware planning system that turns schedules, tasks, and wake-time constraints into realistic night plans.
- Implemented distraction-control features with Accessibility Service, UsageStatsManager, AlarmManager, and WorkManager.
- Built reflective and productivity-focused flows including Focus Mode, Pomodoro timers, analytics, Recovery Mirror check-ins, and lightweight accountability features.
- Handbook: [PillowTalk Website](https://dasbootu9607.github.io/PillowTalk/#/)

### U2INVEST

`Python` `LangGraph` `DeepSeek-V3` `AkShare` `ChromaDB` `HuggingFace Embeddings` `Docker` `SQLite`

Solo full-stack stock education platform combining structured learning, AI-assisted analysis, mock trading, and document-backed retrieval.

- Built a Coursera-style learning platform with progress tracking for beginner-friendly stock education.
- Created an AI Financial Agent with LangGraph StateGraph and custom tool-calling to analyze market news, quotes, and historical K-line data.
- Architected a high-performance RAG pipeline with ChromaDB and MD5-based deduplication for financial PDFs and unstructured JSON.
- Delivered production readiness with Docker, SQLite, and persistent conversation memory.

### ReUnion

`FastAPI` `Streamlit` `LangChain` `ChromaDB` `RAG`

Co-led the backend for a unified career platform that transforms fragmented employment data into personalized roadmaps, job filtering, application tracking, and stack analysis.

- Architected a modular system with an asynchronous FastAPI backend and reactive Streamlit frontend.
- Optimized the RAG kernel by separating ingestion, vectorization, and agentic logic for stronger retrieval quality and faster responses.
- Integrated Gemini CLI into the workflow to accelerate file operations and debugging under hackathon time constraints.

### AIDE

`Python` `Streamlit` `Telegram Bot` `SentenceTransformers` `Ollama` `Llama 3` `ChromaDB`

Team lead and core developer for a four-agent AI ecosystem built to support professional onboarding, learning, and career growth.

- Designed a multi-agent orchestration layer where a Concierge Agent routes user requests across specialized agents.
- Built the Onboarding Agent with RAG over PDF, DOC, and PPT sources while preserving privacy through local Ollama and Llama 3 deployment.
- Established a shared ChromaDB memory layer to support scalable, always-available mentorship experiences.
- Reached Semi-Finalist status at the NTU CAO x SAP Hackathon 2025.
